# Сокращение ссылок через clc.li API

## Как запустить скрипт

Скрипт позволяет:

- сокращать ссылки  
- получать количество кликов по короткой ссылке  

---

## 1. Клонируйте репозиторий

```bash
git clone <repo_url>
cd <repo_folder>
```

---

## 2. Создайте виртуальное окружение

```bash
python3 -m venv venv
```

Активируйте окружение:

```bash
source venv/bin/activate
```

---

## 3. Установите зависимости

```bash
pip install -r requirements.txt
```

---

## 4. Создайте файл .env

В корне проекта создайте файл `.env` и добавьте:

```env
CLC_API_TOKEN=ваш_api_токен
```

---

## 5. Запустите скрипт

Сокращение ссылки:

```bash
python main.py https://example.com
```

Получение количества кликов:

```bash
python cli.py https://clc.li/abc123
```

---

## Ожидаемый результат

Если передана обычная ссылка:

```text
Короткая ссылка: https://clc.li/abc123
```

Если передана короткая ссылка:

```text
Количество кликов: 42
```
